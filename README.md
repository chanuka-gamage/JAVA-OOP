# JAVA-OOP
# Java OOP Project

## 📌 Description

This project is developed using the **Java programming language** and demonstrates the main **Object-Oriented Programming (OOP)** concepts.

The project helps understand how Java uses classes and objects to build structured and reusable programs.

## ⚙️ Requirements

* Java JDK (8 or higher)
* IDE or Code Editor (IntelliJ IDEA, Eclipse, VS Code, NetBeans)
* Operating System: Windows / Linux / macOS

##

## ▶️ How to Compile

Use the following command in the terminal:

```
javac Main.java
```

## ▶️ How to Run

After compiling, run the program using:

```
java Main
```

## 📖 OOP Concepts Used

* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Constructors
* Method Overloading
* Method Overriding

## 🎯 Purpose

The purpose of this repository is to practice **Object-Oriented Programming concepts in Java** and improve software design and coding skills.
